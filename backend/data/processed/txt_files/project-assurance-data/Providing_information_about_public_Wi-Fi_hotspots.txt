How to Find QuickConnect Public Wi-Fi Hotspots
Customers can locate QuickConnect public Wi-Fi hotspots through the following methods:

Using the QuickConnect mobile app which has a built-in hotspot locator.
Visiting the QuickConnect website and using the hotspot map feature.
Looking for the QuickConnect Wi-Fi stickers at partnering locations.



How to Connect to a QuickConnect Public Wi-Fi Hotspot

Ensure the Wi-Fi feature on the device is turned on.
Search for available networks and select the network named "QuickConnect_WiFi."
Open a web browser and you will be redirected to a login page.
Enter your QuickConnect username and password, then click "Login."
Upon successful login, you will have internet access.



Troubleshooting Connection Issues



Issue
Possible Solution




Unable to find QuickConnect_WiFi network
Check if the Wi-Fi is switched on and within range of a hotspot. Restart the device and search again.


Cannot log in
Verify that the username and password are correct. If forgotten, use the password recovery option on the login page or contact customer support.


Weak signal or slow connection
Ensure there are no physical obstructions and try moving closer to the hotspot. If the issue persists, report it to customer support.





Security Concerns and Safe Browsing Practices
Using public Wi-Fi comes with inherent security risks. Here are some tips to ensure safe browsing:

Avoid accessing sensitive information such as online banking or confidential work documents.
Use a VPN to encrypt your internet connection.
Ensure that websites you visit use HTTPS for a secure connection.
Avoid entering personal information on unfamiliar websites.
Keep your device's firewall and antivirus software up to date.
