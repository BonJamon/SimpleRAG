Explaining Call Forwarding Options
Call forwarding is a feature that allows customers to redirect incoming calls to another phone number. Here are the options and instructions for setting up call forwarding:


Option
Description
Activation Code
Deactivation Code


Unconditional Call Forwarding
Forward all incoming calls to another phone number.
*21*[number]#
#21#


Call Forwarding When Busy
Forward calls to another number when the primary line is busy.
*67*[number]#
#67#


Call Forwarding When Not Answered
Forward calls to another number if there is no answer after a predefined number of rings.
*61*[number]#
#61#


Call Forwarding When Unreachable
Forward calls to another number if the primary phone is turned off or out of coverage.
*62*[number]#
#62#


Setting Up Call Forwarding

Choose the appropriate call forwarding option based on the customer's needs.
Instruct the customer to dial the respective Activation Code followed by the phone number to forward calls to, and then #.
If the customer needs to deactivate call forwarding, instruct them to dial the respective Deactivation Code.

Example
If a customer wants to forward all their calls to (123) 456-7890 unconditionally, they would dial:
*21*1234567890#
To deactivate, they would dial:
#21#
