How do I fix connectivity issues?


Restart the primary node and modem.
Ensure nodes are within range of each other.
Update firmware through the mesh network app.
Check for interference from other devices or obstacles.




How can I add more nodes to my network?


Open the mesh network app.
Select the option to add a new node.
Follow the in-app instructions to complete the setup.




How do I manage devices connected to my mesh network?


Open the mesh network app.
Navigate to the device management section.
View connected devices and manage their settings (e.g., prioritize bandwidth, block devices).
