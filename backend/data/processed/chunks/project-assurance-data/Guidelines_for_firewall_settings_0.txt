Guidelines for Firewall Settings
Ensuring that your firewall settings are configured correctly is crucial for maintaining network security while optimizing connectivity. This guide will help you support customers with their firewall settings.
Basic Firewall Configuration
Step-by-Step Instructions:

Access the firewall settings through the router's admin interface.
Log in using admin credentials.
Navigate to the firewall settings menu.
Enable the firewall if it is currently disabled.

Ensure that it is set to a “medium” or “high” security level if adjustable.

Save changes and restart your router.

Important Security Practices
Always remember these security considerations when helping customers:

Use strong passwords for router and firewall access.
Regularly update the router firmware for the latest security patches.
Disable remote management unless necessary.
Regularly back up configuration settings.
