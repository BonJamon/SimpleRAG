Enable Two-Factor Authentication:

            In the Security section, find the Two-Factor Authentication (2FA) option. Click the "Enable 2FA" button.

        

Choose Authentication Method:

            You will be prompted to choose an authentication method. Choose between using a mobile authentication app (e.g., Google Authenticator) or receiving codes via SMS.

        

Set Up the Authentication Method:

            Follow the on-screen instructions to set up the chosen authentication method:

            
Mobile App: Scan the QR code provided with your mobile authentication app, then enter the verification code generated by the app.
SMS: Enter your mobile phone number to receive a verification code via SMS, then enter the code on the QuickConnect website.



Verify and Confirm:

            Once the authentication method is set up, enter the code generated by the app or received via SMS to verify the setup. Click "Confirm" to complete the process.

        

Troubleshooting:



Issue
Solution




Not receiving SMS verification code
Ensure your mobile number is correct and has good network coverage. Try resending the code after a few minutes.
