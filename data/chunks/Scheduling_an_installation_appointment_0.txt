Scheduling an Installation Appointment
This document provides guidance for customer service agents at QuickConnect on how to assist customers with scheduling an installation appointment.
Steps to Schedule an Installation Appointment

Gather Customer Information
Check Availability
Confirm Appointment Details
Send Confirmation

1. Gather Customer Information
Ensure you have the following details from the customer:

Full Name
Contact Number
Email Address
Installation Address
Preferred Time and Date

2. Check Availability
Use the scheduling system to find available slots for the installation. Follow these steps:

Log into the scheduling system
Enter the customerâ€™s postal code
Select the type of installation service required
Review available time slots

3. Confirm Appointment Details
Once a suitable time slot is found, confirm the appointment details with the customer:

Preferred Date and Time
Address
Contact Information
Any special instructions or notes for the technician

4. Send Confirmation
After confirming the appointment, send a confirmation to the customer via email or SMS. The confirmation should include:

Date and Time of the appointment
Technician's name (if available)
Customer Service Contact information
