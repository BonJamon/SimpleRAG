QuickConnect Customer Support: Email Setup Guide
Introduction
This guide is designed to help QuickConnect customer service agents assist customers with setting up their email. Follow the steps outlined below to ensure a smooth and efficient setup process for the customers.
Step-by-Step Email Setup Instructions
1. Collect Necessary Information

Email address
Password
Incoming mail server (IMAP/POP3)
Outgoing mail server (SMTP)
Port numbers for incoming and outgoing servers

2. Entering Email Settings in Email Client
Depending on the email client the customer uses (e.g., Outlook, Thunderbird, or a mobile email app), guide them through entering the gathered information:



Setting
Details




Email Address
The customer's full email address


Password
The customer's email account password


Incoming Mail Server
mail.quickconnect.com (Example for IMAP)


Port Number (IMAP)
993 (SSL)


Port Number (POP3)
995 (SSL)


Outgoing Mail Server
smtp.quickconnect.com


Port Number (SMTP)
465 (SSL) or 587 (TLS)


Use Authentication
Yes (Use the same password as the incoming mail server)



3. Testing the Configuration

Send a test email from the configured email account to ensure outgoing mail is working.
Check the inbox to confirm that incoming mail is properly received.

4. Troubleshooting Common Issues

Authentication Error: Verify the email address and password entered are correct.
Unable to Send Emails: Ensure the outgoing mail server settings are correct and the appropriate port is selected (465 for SSL or 587 for TLS).
Unable to Receive Emails: Double-check the incoming mail server settings and ensure the port numbers match those of
