Resolving Wi-Fi Connectivity Issues

Use this guide to assist customers who are experiencing Wi-Fi connectivity problems. Follow the steps below to troubleshoot and resolve the issue.

Basic Troubleshooting Steps

  • Check the Wi-Fi Signal
    • Ensure the router is powered on and that the Wi-Fi indicator light is glowing.
    • Move the device closer to the router to eliminate any potential signal interference.
  • Restart the Router and the Device
    • Turn off the router, wait for 10 seconds, and then turn it back on.
    • Restart the device that is having connectivity issues.
  • Check Network Settings
    • Ensure the device is connected to the correct Wi-Fi network and that the password is entered correctly.
    • Forget the network and reconnect using the correct login credentials.

Advanced Troubleshooting Steps

  • Update Firmware and Drivers
    • Check for and install any updates for the router's firmware.
    • Update the device's network adapter drivers, if applicable.
  • Change Wi-Fi Channel
    • Log in to the router's admin panel.
    • Navigate to the wireless settings section and change the channel to a less congested one.
  • Reset Network Settings
    • Reset network settings on the device experiencing issues (Note: This will remove all saved networks and credentials).
    • Consult the device's manual for specific instructions on how to reset network settings.
